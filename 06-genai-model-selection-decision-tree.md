# Gen AI Model Selection Decision Tree

## Content type

1. Text Generation
   1. General purpose model - creative writing/conversation
   2. Fine-tuned model - legal, financial, compliance content
   3. Domain-specific model - technical or scientific language
2. Image Generation (same as Text Generation)
3. Audio Generation (same as Text Generation)
4. Video Generation (same as Text Generation)
5. Multimodal Generation (same as Text Generation)
6. Other:
   1. Performance and latency
   2. Customization
   3. Constraints and Resources
   4. Governance, Risk, and Compliance
